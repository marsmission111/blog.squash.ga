Simple WebPage Loader

creating a simple jQuery preloader

Posted by Ryan Curry on December 31, 2018

Creating a loader for your personal website should be simple and easy, and it is! Today, I’ll be walking through how to create a loader from your webpage, from scratch! Though knowledge in CSS and HTML isn’t required for this tutorial, it is highly recommended, as making a personalized loader requires knowledge in this area. With that out of the way, let’s get started.

Creating the Loader

For now, we will have our loader be a simple black overlay. To do this, we will need to create a div element right before the end of the closing body tag. We will need to set the id of the div to “preloader” as you can see below:

<div id=”preloader”></div>

Awesome! Now all we need to do is to style it. In the head tag of your HTML file put a style tag and type in what you see below:

<style>
#preloader {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background: #050505;
  z-index: 800;
  height: 100%;
  width: 100%;
}
</style>

Fading out the Loader

As you can see, the preloader does not fade away, we will need to fix this with jQuery. In between the preloader div and the ending body tag, enter in a script tag and input this code:

<script>
$(document).ready(function() {
  $('#preloader').fadeOut();
});
</script>

Summary
